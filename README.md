# Portfolio

A minimalistic developer portfolio built with Next.js and Tailwind CSS.

## Tech Stack

- **Framework:** Next.js 15
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **AI:** LangChain & Llama 3

## Getting Started

```bash
npm install
npm run dev
```

## Features

- Terminal-inspired UI
- AI Chatbot Assistant
- RAG-powered Q&A
